# 🚀 ProjectPilot-AI

ProjectPilot-AI is an AI-powered project management platform that helps students, developers, freelancers, and small teams efficiently plan, organize, and manage projects. It uses AI to generate project roadmaps and task suggestions, making project execution faster and more organized.

---

## 📌 Problem Statement

Managing projects manually often leads to poor planning, missed deadlines, and inefficient task distribution. Existing tools can be complex or lack intelligent assistance.

ProjectPilot-AI solves this by providing an AI-powered workspace where users can create projects, organize tasks, generate intelligent roadmaps, and track project progress from one place.

---

## ✨ Features

- 🔐 User Authentication
- 📁 Create & Manage Projects
- 🤖 AI-Generated Project Roadmaps
- ✅ Smart Task Suggestions
- 📋 Task Management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/pranjali-mali-ai/ProjectPilot-AI.git
```

### Navigate to Backend

```bash
cd Backend
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Flask Server

```bash
python app.py
```

Open the frontend in your browser or configure Flask to serve it.
